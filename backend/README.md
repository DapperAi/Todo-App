## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Needed dependecy
1. Install Redis :https://redis.io/docs/latest/operate/oss_and_stack/install/install-redis/
2. Start redis, on default port : 6379
